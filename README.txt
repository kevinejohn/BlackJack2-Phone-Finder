BlackJack II Phone Finder

Running Windows Mobile 6.1

Author: Kevin Johnson
Email: kevinejohn@gmail.com


The following files must be in directory:
 \Program Files\GPSSMS\
			PhoneLocator.exe
			SoundAlert.exe
			PhoneStatus.exe
			CallBack.exe
			PhoneRestart.exe


Send a SMS text message with the user defined key and one or more of the following keywords:
  gps         - Sends back SMS with GPS position
  alert       - Plays audio file and displays a "lost phone" message
  status      - Sends back SMS with current battery and reception info
  callmeback  - Calls back the senders phone number
  restart     - Restarts Phone
